        S ::= aAc | bBd | cCa

Conjuntos FIRST e FOLLOW


FIRST(S) = {a, b, c}
FIRST(A) = {a, d}
FIRST(B) = {b, ε}
FIRST(C) = {c, d}
FIRST(D) = {d, a}
                

FOLLOW(S) = {a, c, $}
FOLLOW(A) = {a, c}
FOLLOW(B) = {a, c, d}
FOLLOW(C) = {a}
FOLLOW(D) = {a, c, d}
